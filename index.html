Knowledge Base

Structured for Ongoing Growth

The website now sits on top of a markdown knowledge base. That means new papers, concepts, and relationships can be added in a structured way instead of manually editing every visual element.

Raw Sources

Paper records and source notes live in a separate input layer, which keeps the provenance of each node clear.

Wiki Pages

Each paper has a markdown page with metadata, summary, impact, methods, and graph connections.

Build Script

A small compiler reads the wiki pages and generates the graph dataset used by the public website.

Map Logic

Read the Graph in Three Ways

Use the same graph to understand research evolution over time, topic clusters, and method transitions. The first version focuses on your published and preprint first-authored work.
