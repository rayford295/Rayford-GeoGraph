# Yifan Research Map

This repository contains a public GitHub Pages prototype for an interactive research knowledge graph built around Yifan Yang's first-authored publications.

## Concept

Instead of listing papers as isolated entries, this site presents them as connected nodes in a research map. Each node highlights:

- the paper title and venue
- the core research themes
- the methods used
- links to the paper and code
- how the paper connects to earlier and later work

## Initial Scope

The first release focuses on first-authored publications and preprints already visible on the main academic website:

- GeoLocator
- Hyperlocal Disaster Damage Assessment
- Perceiving Multidimensional Disaster Damages
- DamageArbiter
- Satellite-to-Street
